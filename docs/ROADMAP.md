# Alika's Wardrobe

Mission:

Build the world's best AI-powered personal wardrobe and styling companion.

The app should evolve from a structured digital wardrobe into a photorealistic virtual dressing room capable of styling, planning, packing and generating editorial-quality lookbooks using real wardrobe items.

Every future feature should support this mission.

## Philosophy

- Image-first, always.
- Premium editorial design over generic utility UI.
- The product should feel like a luxury dressing room, not inventory software.
- The wardrobe is the single source of truth.
- Real wardrobe items only. No invented styling data unless clearly marked.
- Creative workflows should come before admin workflows.
- AI should enhance styling decisions, not replace personal taste.
- Every interaction should feel calm, elegant, and enjoyable.

## Development Principles

- Build stable foundations first.
- Avoid rebuilding completed work.
- Every new feature should integrate naturally with existing workflows.
- Maintain premium editorial design language.
- Image-first UX.
- Keep interactions enjoyable.
- Avoid spreadsheet-style interfaces wherever possible.

## Working Rule For Future Development

Future Codex sessions should read this roadmap before implementing major features so development stays aligned with the long-term vision.

This document should evolve with the project.

It should continue to describe where Alika's Wardrobe is going, why it exists, and how new phases should support the mission rather than fragment it.
